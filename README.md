# Sales Analysis Dashboard (Power BI)

## Project Overview

This project presents an interactive Sales Analysis Dashboard developed using Microsoft Power BI. The dashboard helps analyze business performance by providing clear insights into sales, profit, product performance, and trends.

---

## Objectives

* Analyze overall sales and profit performance
* Identify top-performing products
* Track monthly sales trends
* Provide interactive filtering for better decision-making

---

## Features

* KPI Cards (Total Sales, Profit Percentage, Quantity Sold)
* Monthly Sales Trend Analysis
* Product-wise Sales Distribution
* Payment Mode Analysis (Donut Chart)
* Interactive Slicers (Month and Year)
* Clean and user-friendly dashboard design

---

## Tools and Technologies

* Microsoft Power BI
* Data Visualization Techniques
* Basic DAX (Data Analysis Expressions)

---

## Key Insights

* Identified high-performing products contributing to maximum revenue
* Observed monthly trends for better forecasting
* Analyzed customer payment preferences
* Improved decision-making using interactive filters

## Conclusion

This project demonstrates the effective use of Power BI for business intelligence and data visualization, transforming raw data into meaningful insights.
